## AI Dataset Quality Inspector

An open-source AI tool for automatic dataset quality analysis, including:

- Missingness detection
- Bias analysis
- Distribution drift detection

### Why this matters
Data quality issues silently break ML systems before training even begins.

### Features
- Model-agnostic
- Explainable outputs
- Research-friendly
- OpenML compatible

### Roadmap
- Integrate with openML datasets
- Add fairness metrics
- Extend drift detection
